# 📚 Book Bot - AI Book Recommendation Chatbot

A conversational AI chatbot built with Python that recommends books based on your preferences.

## Tech Stack
- Python
- Google Gemini API
- Flask
- HTML, CSS, JavaScript

## Features
- AI-powered book recommendations
- Clean chat interface
- Powered by Google Gemini

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Set your API key: `export GEMINI_API_KEY="your-key"`
4. Run the app: `python3 app.py`
5. Open `http://localhost:5000` in your browser
